# <a name="office-365-client-app-support--single-sign-on"></a>Office 365-Client-App-Unterstützung – einmaliges Anmelden

Einmaliges Anmelden (Single Sign-on, SSO) fügt Sicherheit und Bequemlichkeit hinzu, wenn sich Ihre Benutzer bei Anwendungen in Azure Active Directory (Azure AD) anmelden. Bei einmaligem Anmelden melden sich die Benutzer mit einem Konto einmal an, um auf Domänen verbundene Geräte, Unternehmensressourcen, Software as a Service (SaaS)-Anwendungen und Webanwendungen zuzugreifen.

Weitere Informationen finden Sie [unter Single Sign-on](https://docs.microsoft.com/azure/active-directory/manage-apps/what-is-single-sign-on).

## <a name="supported-platforms"></a>Unterstützte Plattformen

 - Windows 10-Desktop
 - Moderne Windows 10-apps
 - Webbrowser
 - Android
 - IOS<sup>1</sup>
 - macOS

Weitere Informationen zur Plattformunterstützung in Office 365 finden Sie unter [System Requirements for Office 365](https://products.office.com/office-system-requirements).
