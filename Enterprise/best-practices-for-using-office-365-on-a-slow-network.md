# <a name="best-practices-for-using-office-365-on-a-slow-network"></a>Bewährte Methoden für die Verwendung von Office 365 bei langsamen Netzwerken

Wäre es noch etwas, wenn Ihre Internet-Verbindung wurde immer schneller und nie nach-unten? Möglicherweise wird dieses Tages stammen. Aber in der Zwischenzeit praktische Dinge, die Sie tun können, um ein balky Netzwerk zu umgehen und weiterhin Erledigung Ihrer tägliche Arbeit vorhanden sind. Obwohl Office 365 eine Cloud-basierten Dienst ist, bietet aber auch unterschiedlichste Weise durch Ihren Inhalt offline arbeiten und die Änderungen synchronisiert halten. Es ist auch in einigen Fällen weitere effiziente Inhalt offline entwickelt, nur weil Applications schneller ausgeführt und die Benutzeroberfläche mehr reagiert. Der Punkt ist dies: Office 365 bietet Ihnen die Vorteile beider. Hier ist wie das nutzen. 
  
> [!TIP]
> Möchten Sie, wie langsam (oder fast) angezeigt, dass die Verbindung zum Netzwerk ist? Versuchen Sie die [Geschwindigkeit der OOKLA testen](https://www.speedtest.net/) oder das [Netzwerk Geschwindigkeit testen App](https://www.windowsphone.com/en-us/store/app/network-speed-test/9b9ae06b-2961-41ef-987d-b09567cffe70). 
     
## <a name="why-is-my-network-so-slow"></a>Warum wird mein Netzwerk so langsam?

Obwohl Sie nicht über die Kontrolle über die Leistung des Netzwerks selbst verfügen, sollten sie wissen, was im Hintergrund. Das Internet ist enorm komplexen, aber es gibt einige grundlegende Konzepte, die Ihnen das Verständnis der Situation viel besser helfen kann. Befolgen die bewährten Methoden in diesem Artikel können Sie Hilfe zur Umgehung für dieses Problem beheben von Leistungsproblemen und Unzufriedenheit.
  
**Wichtige Faktoren, die die Leistung des Netzwerks beeinträchtigen**

![Netzwerkleistungsfaktoren](media/62a94322-3f1a-4d2d-bbdc-2aa0722d2d96.png)
  
 **Bandbreite und Wartezeit** Die beiden wichtigsten Maßnahmen der Netzwerk-Performance sind Bandbreite und Wartezeit: 
  
- Bandbreite ist die Rate der Durchsatz gemessen in Bits pro Sekunde. Größer ist besser. Bandbreite ist wie eine Pipe Wasser. Je größer der Pipe, die mehr Wasser, die Sie "ihn über einfügen können".
    
- Latenz ist die Zeit dauert für Inhalt auf Ihrem Gerät von einem Server oder Dienst abgerufen und in Millisekunden gemessen wird. Schneller ist besser. Wartezeit kann durch eine Reihe von Faktoren beeinflusst, einschließlich geringer Bandbreite, wenige Verbindung oder Übertragungszeit verursacht werden.
    
 **Häufige Probleme** Neben Bandbreite und Wartezeit andere Probleme auf die Leistung des Netzwerks auswirken und sind oft unvorhersehbar. Netzwerk-Performance kann basierend auf dem Zeitpunkt der Tag oder Ihren physischen Standort schwanken. Das Netzwerk kann bei bestimmten Ereignissen, die die Verwendung des Internets, wie Naturkatastrophen oder ein wichtiges öffentliche Ereignis zunehmen beeinträchtigen. Die Größe und Komplexität der Seite geladen wird und die Anzahl und Größe der übertragenen Dateien haben einen direkten Einfluss auf die Leistung. Eine WLAN-Verbindung vorübergehend beeinträchtigen kann: Angenommen, Sie eine große Konferenz Tausende von jeder gleichzeitig Tweet anfordern abrufen. 
  
 **Überlegungen für ein Netzwerk Satelliten** Ein Netzwerk Satelliten ist nützlich, wenn ein Land Netzwerk nicht möglich, wie das rückverweisende Land, eine Cruise liefern oder ein remote wissenschaftliche Bereich ist. Diese Netzwerke nutzen Satelliten in eine geosynchronous Orbit 22.000 Meilen oberhalb der Äquator positioniert. Jedoch eine Übertragung Grundsätze tatsächlich etwa 90.000 Meilen und so verfügt über ein Netzwerk Satelliten eine langsamere Wartezeit (500 ms oder mehr) als ein Land Netzwerk (20 auf 50 ms). Klicken Sie unter am besten der Bedingungen können Sie diese Wartezeit nicht beachten, doch für große Dateien herunterladen, streaming von Videos und spielen, werden Sie wahrscheinlich. Ein weiteres Problem ist "Verblassen Regen" in welche extra fette Wetter, wie Gewittern und Blizzards, vorübergehend Satelliten Übertragung unterbrechen kann.
  
## <a name="are-you-sure-its-the-network"></a>Sind Sie sicher, dass das Netzwerk ist?

Wenn Leistungsprobleme auftreten, stellen Sie zunächst sicher, dass das Gerät nicht die Ursache des Problems ist. Es gibt zwei Dinge, die Sie ausführen können, die eine große Verbesserung vornehmen können:
  
- Stellen Sie sicher, dass Ihr Gerät wird ebenfalls ausgeführt und keine Malware auf Ihrem Computer vorhanden ist.
    
- Kaufen Sie wenn möglich, mehr Speicher. Hinzufügen von Arbeitsspeicher ist die einfachste und häufig die meisten effektiv zum Verbessern der Leistung auf dem Gerät. Es ist besonders hilfreich beim Arbeiten mit großen Dateien und Videos.
    
Weitere Informationen finden Sie unter [Windows-Leistung und Wartung](https://windows.microsoft.com/en-us/windows/performance-maintenance-help#performance-maintenance-help) und [Windows Update Systemprobleme Leistung](https://support.microsoft.com/mats/slow_windows_performance/).
   
## <a name="best-practices-for-using-your-browser"></a>Bewährte Methoden für die Verwendung Ihres Browsers

Ihr Browser ist Ihr Gateway zu Office 365, sodass es auf die Leistung, insbesondere im Hinblick auf die Zeit auswirken kann, die zum Laden benötigt eine Seite und wie oft Sie runden zu bringen das Office 365-Dienst. 
  
 **Im Allgemeinen Browser**
  
Hier finden Sie einige Vorschläge für Browser im Allgemeinen:
  
- Deaktivieren Sie Browser-Add-ons, die möglicherweise die Leistung beeinträchtigen oder, die nicht wirklich erforderlich sind.
    
- Erhöhen Sie die Cachegröße für die temporäre Internetdateien.
    
-  Nachdem Sie sich bei Ihrer Arbeit oder Schule Konto angemeldet haben, lassen Sie das Browserfenster geöffnet im Verlauf des Tages. Sie können anderen Registerkarten und Windows öffnen, ohne sich erneut anzumelden. Wenn Sie an ein anderes Konto anmelden müssen, verwenden Sie Private durchsuchen. 
    
- Nachdem jeder Seite heruntergeladen und geöffnet ist, lassen sie mithilfe der Registerkarten geöffnet. Es ist einfach zu navigieren zwischen den Registerkarten und verwenden Sie die Seite später in den Tag. Aktualisieren Sie eine Seite nur, wenn Sie die neuesten Daten auf dieser Seite benötigen.
    
- Wenn eine Seite lange dauert, Beenden des Seite Downloads (drücken Sie ESC), und klicken Sie dann aktualisieren Sie die Seite (drücken Sie F5). 
    
-  Wenn möglich, reduzieren Sie Roundtrips zu Office 365. Beispielsweise statt Paging für Listen oder Bibliotheken, Suchen mithilfe von Dateien in einer großen Bibliothek zu suchen und Filtern in einer Liste direkt auf die Ergebnisse abgerufen werden soll. Oder Erstellen von Ansichten, die Ladezeit der Seite zu minimieren. Weitere Informationen finden Sie unter [Verwalten umfangreicher Listen und Bibliotheken in Office 365](https://support.office.com/article/b4038448-ec0e-49b7-b853-679d3d8fb784#BKMK_PAGES).
    
- Wenn video Leistung schlecht ist, können Sie möglicherweise auf das Video herunter, und beobachten sie, wie auf dem Gerät. Download-Verknüpfung ist möglicherweise verfügbar, oder klicken Sie mit der rechten Maustaste auf den Link video, und wählen **Ziel speichern als**können Sie möglicherweise. 
    
 **Browser-spezifischen**
  
Hier sind einige Vorschläge für bestimmte Browser:
  
- **InternetExplorer** Eine Aktualisierung auf Internet Explorer Version 11 oder höher für erhebliche Leistungssteigerungen gegenüber früheren Versionen. Weitere Informationen finden Sie unter [Beheben von Internet Explorer-Probleme ](https://support.microsoft.com/mats/ie_performance_and_safety).
    
- **FireFox** Weitere Informationen finden Sie unter [Firefox ist langsam oder reagiert nicht mehr](https://support.mozilla.org/en-US/products/firefox/fix-problems/slowness-or-hanging).
    
- **Safari** Weitere Informationen finden Sie unter [Apple - Safari](https://www.apple.com/safari/).
    
- **Chrome** Weitere Informationen finden Sie unter [Chrome-Hilfe](https://support.google.com/chrome/?hl=en).
  
## <a name="best-practices-for-using-outlook-and-outlook-web-app"></a>Bewährte Methoden für die Verwendung von Outlook und Outlook Web App

Lesen, schreiben und zum Organisieren von e-Mail ist ein wichtiger Teil der jeder Tag. Outlook und Outlook Web App (OWA) bieten offline-Unterstützung. Mithilfe einer e-Mail-app auf Ihrem Smartphone ist eine andere nützliche Alternative. Verwenden Sie die folgenden Optionen, die am besten für Ihre Bedürfnisse:
  
- Aktualisieren Sie auf Outlook 2013 SP1 oder höher für erhebliche Leistungssteigerungen gegenüber früheren Versionen. 
    
-  Outlook Web App ermöglicht die Erstellung offline-Nachrichten, Kontakte und Kalenderereignisse, die hochgeladen werden, wenn OWA nächsten ist eine Verbindung zu Office 365 herstellen. Weitere Informationen zur Einrichtung und Verwendung von OWA im Offlinemodus finden Sie unter [Verwenden von Outlook Web App offline](https://support.office.com/article/3214839c-0604-4162-8a97-6856b4c27b36).
    
- Outlook ermöglicht Ihnen im Cache-Modus, in dem er automatisch nach Möglichkeit eine Verbindung herstellt. Sie können Outlook das gesamte Postfach oder nur einen Teil davon herunterladen müssen. Weitere Informationen finden Sie unter [Aktivieren der Exchange-Cache-Modus](https://support.office.com/article/7885af08-9a60-4ec3-850a-e221c1ed0c1c) und [offline arbeiten in Outlook](https://support.office.com/article/f3a1251c-6dd5-4208-aef9-7c8c9522d633). 
    
- Outlook bietet außerdem einen offline-Modus. Um dies zu verwenden, müssen Sie zuerst-Cache-Modus einrichten, damit Informationen von Ihrem Konto nach unten zu Ihrem Computer kopiert wird. Outlook versucht im Offlinemodus, das eine Verbindung über das Senden und Empfangen von Einstellungen, oder wenn Sie manuell festlegen, dass online arbeiten. Weitere Informationen finden Sie unter [zur Vermeidung von Gebühren für die Daten offline arbeiten](https://support.office.com/article/827fe51f-5609-4062-82b4-3578057f9282), [Ändern Sie senden und Empfangen von Einstellungen, wenn Sie offline arbeiten](https://support.office.com/article/f681ec10-cb14-40cb-8709-1909a13c304a)und [Switch offline zu online arbeiten](https://support.office.com/article/2460e4a8-16c7-47fc-b204-b1549275aac9). 
    
- Wenn Sie über ein Smartphone verfügen, können Sie es auf Ihre e-Mails und Kalender über Netzwerk des Mobilfunkbetreibers selektieren verwenden. 
    
> [!NOTE]
> Nachfolgend finden Sie einige Hinweise dazu, wann Outlook oder OWA verwendet werden. Wenn Speicherplatz nicht um ein Problem auf Ihrem Gerät ist, Outlook verfügt über eine umfassende Auswahl an Features und möglicherweise am besten für Sie geeignet. Speicherplatz auf dem Gerät ein Problem ist, sollten Sie OWA verfügt über eine Teilmenge der Features, aber auch am besten in einer Situation online verwenden. Selbstverständlich können Sie entweder verwenden, da sie gut zusammenarbeiten. 
  
## <a name="best-practices-for-using-onedrive-for-business"></a>Bewährte Methoden für die Verwendung von OneDrive für Unternehmen

OneDrive für Unternehmen wird von Grund auf neu entwickelt, mit Ihrer Dateien online und offline arbeiten. Nachdem Sie einrichten Synchronisierung von Änderungen erfolgt automatisch und zuverlässig stellen jederzeit und überall Sie diese. Wenn das Netzwerk langsam ist, können Sie mit der offline-Version der Dateien arbeiten.
  
Die OneDrive for Business Sync app ist in Office 2013 (Professional Plus oder Standard Edition) oder ein Office 365-Abonnement, die Office 2013-Anwendungen zur Verfügung. Wenn Sie nicht über Office 2013 verfügen, können Sie die OneDrive for Business Sync app [herunterladen](https://support.microsoft.com/kb/2903984) kostenlos. Diese app ist außerdem schneller als mit den Befehlen **im Explorer öffnen** oder **Hochladen** . Weitere Informationen finden Sie unter [Einrichten des Computers Synchronisieren Ihrer OneDrive for Business-Dateien in Office 365](https://support.office.com/article/23e1f12b-d896-4cb1-a238-f91d19827a16).
  
Hier finden einige zusätzliche Richtlinien für die mit dem OneDrive for Business Sync-app:
  
- Wenn Sie eine große Bibliothek zum ersten Mal synchronisiert sind, starten Sie die Synchronisierung während der Spitzenzeiten beispielsweise über Nacht. 
    
- [Beenden Sie die Synchronisierung einer Bibliothek mit dem OneDrive for Business-app](https://support.office.com/article/a7e41f1f-3a98-4ca7-9443-f10250688330) -Feature können Sie vorübergehend beenden Sie die Synchronisierung von Updates. Verwenden Sie diese Funktion jedoch für eine kurze, wie ein paar Stunden nacheinander ein, um große queuing zu vermeiden Zahlen von Updates und das Risiko von Zusammenführungskonflikte zu minimieren, wenn mehrere Personen am selben Dokument arbeiten. 
  
## <a name="best-practices-for-using-onenote"></a>Bewährte Methoden für die Verwendung von OneNote

Alle SharePoint-Teamwebsite verfügt über integrierte OneNote-Notizbuch, und Sie können auf einfache Weise eigene erstellen. OneNote ist eine hervorragende Möglichkeit zum angemessenen Informationen zu sammeln, benötigen Sie jeden Tag und Aufgaben auszuführen. Beispielsweise verwenden viele Teams OneNote als eine Sammelstelle für wöchentliche Besprechungen, Projektnotizen, Ideen, Pläne und Statusberichte. Sie können diese unterschiedlichen Informationen übersichtlich organisieren, mithilfe von Registerkarten, Abschnitte und Seiten.
  
Der große Vorteil von OneNote ist jedoch, dass Sie den Inhalt von praktisch jedem Gerät, ob ein Desktop, einen Laptop, einem Tablet oder einem Smartphone zugreifen können. Und Sie müssen keinen Gedanken speichern oder synchronisieren, da OneNote für Sie erledigt. 
  
Weitere Informationen finden Sie unter [Microsoft OneNote](https://office.microsoft.com/en-us/onenote/).

## <a name="best-practices-for-using-project-online"></a>Bewährte Methoden für die Verwendung von Project Online

Die folgenden Richtlinien können Sie netzwerkleistung verbessern.
  
- Project Online und SharePoint Online erfordern Synchronisierung, die zeitaufwendig sein kann. Wenn die Projektteams niedrig Umsatz haben, deaktivieren Sie die Projektwebsite-Synchronisierung zum Verbessern der Leistung Projekt zu veröffentlichen und Projektdetailseiten verwalten. Active Directory-Synchronisierung an eine Gruppe von Ressourcen, die tatsächlich müssen, verwenden Sie das System und überwachen potenzielle Probleme beim Berechtigung nach der Synchronisierung des große Gruppen zu begrenzen. 
    
- Wenn Ihre Organisation Projektwebsites verwendet, erstellen sie bei Bedarf statt automatisch. Dies beschleunigt den ersten Veröffentlichung Eindruck und vermeidet unnötige Websites und Inhalte erstellen.
    
- Project Detail Pages (PDP) eine neuberechnung des gesamten Projekts auslösen und Kickoff Workflowaktionen, die beide rechenintensive Vorgänge werden können. Um zu vermeiden, zwei Update-Prozesse zur selben Zeit auf die gleiche PDP auslösen, nicht die Kalender (Anfangstermin, Endtermin, Statusdatum und aktuelles Datum) sowie der Felder nicht geplanten (Projektname, Beschreibung und Besitzer) aktualisieren. 
    
- Reduzieren der Anzahl der Webparts und benutzerdefinierte Felder auf jede PDP angezeigt. Erstellen Sie einen dedizierten PDP mit nur die Felder, die zum Verbessern der Last und sparen Sie Zeit aktualisiert werden müssen. 
    
- Bei Verwendung von OData für reporting beschränken Sie die Menge der Daten, die Sie zur Laufzeit mithilfe von serverseitige Filterung Abfragen.
    
Weitere Informationen finden Sie unter [Project Online Optimieren der Leistung](https://support.office.com/article/12ba0ebd-c616-42e5-b9b6-cad570e8409c).
